
# Card Marketplace

Aplicação web para gerenciamento e troca de cartas, desenvolvida com **Vue 3**, com foco em organização, escalabilidade e boas práticas de arquitetura frontend.

## Tecnologias Utilizadas
- **Vue 3** 
- **TypeScript** 
- **Quasar Framework** 
- **Vue Router** 
- **Pinia** 
- **Axios**
- **Vee Validate** 
- **SCSS (BEM)** 

# Configuração do projeto


```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
